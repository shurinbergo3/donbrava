# Donbrava Emigration - сайт

Современный сайт иммиграционного агентства (переезд в Словакию под ключ). Статический: HTML + CSS + ванильный JS, без сборки. Открывается двойным кликом по `index.html` или с любого хостинга.

## Структура
```
index.html               главная (hero, подбор-квиз, услуги, процесс, видео, отзывы, контакты)
services/vnz-pmz.html    детальная страница услуги «ВНЖ и ПМЖ» (+ FAQ-разметка)
journal/index.html       журнал (список статей с фильтрами + рассылка)
journal/article.html     страница статьи
assets/css/styles.css    дизайн-система и все стили
assets/js/main.js        меню, scroll-reveal, magnetic-кнопки, квиз, FAQ, форма, счётчики
assets/img/              изображения и логотип
robots.txt, sitemap.xml  SEO
```

## Дизайн
- Палитра: глубокий navy + тёплая бумага (cream) + приглушённое золото.
- Шрифты: Playfair Display (заголовки) + Manrope (текст/интерфейс), оба с кириллицей, подключены с Google Fonts.
- Логотип-вордмарк собран типографикой в фирменной палитре; оригинал со старого сайта лежит в `assets/img/logo.png`, монограмма-фавикон - `assets/img/favicon.svg`.

## Что заменить перед продакшеном
1. **Домен.** В `<link rel="canonical">`, og-тегах, JSON-LD и `sitemap.xml` стоит `https://donbrava.sk/` - поменяйте на реальный.
2. **Форма заявки.** Сейчас отправка обрабатывается на стороне браузера (показывает экран «Заявка принята»). Подключите бэкенд/Formspree/Telegram-бота: обработчик в `assets/js/main.js` (`#leadForm`).
3. **Видео.** Превью для блока «Видео-гайды» - кадры Братиславы. При желании замените на реальные обложки роликов с YouTube; ссылки уже ведут на канал.
4. **Фото.** Изображения - Братислава/Татры с Unsplash (свободная лицензия). Можно заменить на собственные (офис, команда, кейсы) - это усилит E-E-A-T.
